D3 Workshop - The Basics

The fundamentals of D3 - the Exercises

And small push to help you cross the initial barrier of enter D3 magic.

What to expect from this Workshop

This is an exercises based introduction to the D3 library. We are gonna focus in the basics of D3 to try to solve the initial possible difficulties. We will do some exercises that will cover enough to allow you to continue solo.

Pre-requisites

Basics understanding of HTML and JS will be enough

How to execute the exercises

The demos of the exercises are listed below.

The JS code is using EMACSScript6.

syntax so you will need to open them using a browser that supports this syntax. I am using Chrome Version 72.0.3626.96.

If you are running the exercises in your local you will need to run them using a web server. The reason is that we are using AJAX to load the CSV data files and AJAX doesn't support to load files directly from your hard drive, it has to be through HTTP.

Disclaimer

Time pressure: I will do my best to complete the workshop in 1 hour. This is why we can not do a real full demo of each exercise. Some times I will try to do some real time code but if I see we are finding difficulties I will jump to the pre-written version

I am not in any way a D3 expert. All the knowledge I am sharing here has been acquired in the last weeks mostly based on try and error and some video tutorials and books. So my code can have bugs. I also tried to follow the code style standards of the D3 community but some mistakes may have happened.
